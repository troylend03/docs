# Rallies Support Center

This repository contains the documentation for [Rallies.ai](https://rallies.ai), an AI-powered investment research platform.

## Structure

```
├── getting-started/     # Onboarding and setup guides
├── account/             # Account management articles
├── billing/             # Subscription and payment help
├── ai-assistant/        # AI Research Assistant guides
├── research/            # Stock pages and screener
├── watchlists/          # Watchlist features
├── portfolio/           # Portfolio tracking
├── data/                # Data sources information
├── mobile/              # Mobile app guides
├── ai-arena/            # AI Arena feature
├── troubleshooting/     # Common issues and fixes
├── legal/               # Disclaimers and privacy
├── support/             # Contact information
└── docs.json            # Mintlify configuration
```

## Development

Install the Mintlify CLI to preview documentation changes locally:

```bash
npm i -g mint
```

Run the following command at the root of the documentation:

```bash
mint dev
```

View your local preview at http://localhost:3000.

## Publishing

Changes pushed to the main branch are automatically deployed via the Mintlify GitHub integration.

## Content Guidelines

- Use second-person voice ("you")
- Keep articles focused and actionable
- Include relevant links to related articles
- Follow existing formatting patterns
