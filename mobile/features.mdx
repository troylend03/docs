# Mobile App Features

The Rallies mobile app brings AI-powered investment research to your pocket. While it offers the same core functionality as the web version, several features are optimized specifically for mobile use.

## Core Features Available on Mobile

Everything you use on the web is available on mobile:

| Feature | Description |
|---------|-------------|
| AI Research Assistant | Ask questions about stocks, markets, and investing |
| Watchlists | Track stocks and monitor price changes |
| Portfolio Tracking | View and analyze your holdings |
| Price Alerts | Set notifications for target prices |
| Stock Pages | Access company data, financials, and analysis |
| Vibe Screener | Find stocks using natural language |
| Deep Research | Generate comprehensive research reports (Pro) |

## Mobile-Exclusive Features

### Push Notifications

Get instant alerts delivered directly to your phone.

**What you can receive:**
- **Price alerts** - Immediate notification when a stock hits your target
- **Breaking news** - Important updates on your watchlist stocks
- **Research complete** - Know when your Deep Research report is ready
- **Account alerts** - Security notifications and billing reminders

Push notifications are faster than email and appear directly on your lock screen.

[Set up price alerts](/watchlists/price-alerts)

### Biometric Authentication

Secure your account without typing passwords.

**Supported methods:**
- Face ID (iPhone X and later)
- Touch ID (older iPhones, some iPads)
- Android facial recognition and fingerprint sensors

[Enable biometric login](/mobile/setup#step-3-set-up-biometric-login)

### Home Screen Widgets

Glance at stock data without opening the app.

**iOS widgets:**
| Size | What It Shows |
|------|---------------|
| Small | Single stock price and daily change |
| Medium | Top 3-5 stocks from a watchlist |
| Large | Portfolio summary with top holdings |

**Android widgets:**
- Watchlist preview with prices
- Portfolio value and daily performance
- Single stock tracker

### Offline Access

View previously loaded data without an internet connection.

**Works offline:**
- Previously viewed stock pages
- Cached portfolio data
- Saved watchlists
- Past AI conversations

**Requires internet:**
- New AI queries
- Real-time price updates
- Portfolio sync

### Voice Input

Tap the microphone icon to speak your question instead of typing. Works best in quiet environments. Say ticker symbols letter by letter for accuracy.

## AI Research Assistant on Mobile

The AI works the same on mobile, with interface optimizations:

- **Full-screen chat** - More room for conversation
- **Quick prompts** - Tap suggested questions to start
- **Voice input** - Speak instead of type
- **Easy sharing** - Share AI responses via text, email, or other apps
- **Contextual chat** - When viewing a stock page, the AI knows which stock you're asking about

## Portfolio Features

### Portfolio Dashboard

The mobile portfolio view shows:
- Total portfolio value
- Today's gain/loss (dollars and percentage)
- Individual position cards
- Allocation charts

**Gestures:**
- Swipe down to refresh
- Tap a position for full details
- Swipe left on a position for quick actions

### Position Details

Tap any holding to see:
- Current value and cost basis
- Gain/loss since purchase
- Stock fundamentals
- Recent news

## Watchlist Features

### Managing Watchlists

- **Create** - Tap + to add a new watchlist
- **Rename** - Tap the edit icon
- **Delete** - Swipe left and tap delete
- **Reorder** - Tap and hold to drag

### Stock Cards

Each stock shows:
- Current price and daily change
- Mini sparkline chart
- Tap to expand for more data

Swipe left on any stock for quick actions: set alert, view page, add to portfolio, or remove.

## Stock Pages on Mobile

Full company information in a mobile-friendly layout:

- **Overview** - Key stats, price, description
- **Financials** - Revenue, earnings, balance sheet
- **Analysis** - Analyst ratings, price targets
- **News** - Recent headlines and filings
- **Charts** - Interactive price charts

### Mobile Chart Features

- **Pinch to zoom** - Focus on specific time periods
- **Tap and hold** - See exact prices at any point
- **Rotate to landscape** - Full-screen chart view

## Comparing Mobile vs. Web

### Better on Mobile

| Feature | Why |
|---------|-----|
| Price alerts | Push notifications are instant |
| Quick price checks | Widgets and quick actions |
| On-the-go research | Full functionality anywhere |

## Accessibility Features

**iOS:**
- VoiceOver compatible
- Dynamic Type support
- Reduce Motion option

**Android:**
- TalkBack compatible
- Font size scaling
- Color inversion support

**In-app:**
- Adjustable chart colors for colorblindness
- Haptic feedback can be disabled in Settings > Accessibility

## Frequently Asked Questions

### Is any data not available on mobile?

All data available on web is available on mobile. Some visualizations are simplified for smaller screens, but no information is missing.

### Can I make trades through the app?

No, Rallies is for research only. Use your brokerage's app to execute trades.

### Do widgets count against my data usage?

Widgets use minimal data and are designed to be battery efficient.

### Can I use the app in landscape mode?

Yes, the app supports landscape orientation. Charts are particularly useful in landscape view.
