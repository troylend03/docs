# Supported Brokerages

Connect your brokerage to Rallies for automatic portfolio syncing. Here's what's supported.

## Fully Supported Brokerages

These brokerages have full integration:

### Major Brokerages
- **Fidelity**
- **Charles Schwab**
- **TD Ameritrade** (now part of Schwab)
- **E*TRADE**
- **Vanguard**
- **Robinhood**
- **Interactive Brokers**
- **Merrill Edge**
- **Webull**

### Other Supported Brokerages
- Ally Invest
- Firstrade
- SoFi Invest
- Public
- M1 Finance
- Wealthfront
- Betterment
- Acorns
- Stash
- Cash App Investing

## What's Synced

From connected brokerages:

| Data | Synced |
|------|--------|
| Stock holdings | ✓ |
| ETF holdings | ✓ |
| Mutual funds | ✓ |
| Options | Varies |
| Crypto (if offered) | Varies |
| Cash balance | ✓ |
| Cost basis | If available |
| Transaction history | Recent |

## Not Yet Supported

We're working on adding support for:

- Some international brokerages
- Certain workplace retirement plans
- Some crypto-only exchanges

**Don't see your brokerage?** You can:
1. [Request we add it →](/support/feedback)
2. [Add holdings manually →](/portfolio/add-manually)

## Connection Requirements

To connect, you'll need:

- **Login credentials** for your brokerage
- **Two-factor authentication** may be required
- **Security questions** answers (if your brokerage uses them)

## Account Types Supported

| Account Type | Supported |
|--------------|-----------|
| Individual brokerage | ✓ |
| Joint accounts | ✓ |
| IRA (Traditional, Roth) | ✓ |
| 401(k) - Personal access | ✓ |
| 401(k) - Employer plans | Limited |
| Custodial accounts | Varies |
| Trust accounts | Varies |

<Note>
Workplace retirement plans (401k, 403b) through employers may have connection limitations depending on the plan provider.
</Note>

## Checking if Your Brokerage Works

1. Go to **Portfolio → Connect Account**
2. Search for your brokerage name
3. If it appears, it's supported
4. If not, [let us know](/support/feedback)

## Multiple Brokerages

You can connect multiple brokerage accounts:

- All holdings are combined in your portfolio view
- Each account is tracked separately
- See breakdown by account or total
- No limit on number of connections

## Sync Frequency

Connected accounts sync automatically:

| Type | Frequency |
|------|-----------|
| Holdings | Every few hours |
| Prices | Real-time (Pro) / 15-min (Free) |
| Transactions | Daily |

You can also manually trigger a sync anytime.

## Why Some Brokerages Aren't Supported

- **Technical limitations** - Some brokerages don't allow third-party connections
- **International regulations** - Different rules in different countries
- **Low demand** - We prioritize most-requested brokerages
- **Security restrictions** - Some institutions block all aggregation

## Request a Brokerage

Want us to add your brokerage?

1. [Submit a feedback request →](/support/feedback)
2. Include your brokerage name
3. We track all requests and prioritize by demand
