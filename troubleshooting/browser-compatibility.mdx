# Browser Compatibility Issues

Rallies works best on modern browsers. If you're experiencing issues, your browser might be the cause.

## Supported Browsers

### Fully Supported
- **Chrome** 90+ (recommended)
- **Safari** 14+
- **Firefox** 90+
- **Edge** 90+

### Limited Support
- Older browser versions
- Internet Explorer (not supported)
- Uncommon browsers

## Common Browser Issues

### Page Won't Load

**Symptoms:**
- Blank white screen
- Infinite loading spinner
- "This site can't be reached"

**Solutions:**
1. **Update your browser** to the latest version
2. **Clear cache and cookies** ([how to →](/troubleshooting/clear-cache))
3. **Try a different browser** to isolate the issue
4. **Disable extensions** that might interfere

### Features Not Working

**Symptoms:**
- Buttons don't respond
- Dropdowns won't open
- Charts not displaying

**Solutions:**
1. **Enable JavaScript** - Rallies requires JavaScript
2. **Disable ad blockers** temporarily
3. **Try incognito/private mode** - Tests without extensions
4. **Update your browser** - Old versions lack features we use

### Display Issues

**Symptoms:**
- Layout looks broken
- Text overlapping
- Elements in wrong positions

**Solutions:**
1. **Zoom to 100%** - Press `Ctrl/Cmd + 0`
2. **Clear cache** - Old styles may be cached
3. **Check window size** - Some features need minimum width
4. **Update browser** - CSS may not render properly on old versions

## Browser-Specific Fixes

### Chrome

**Enable JavaScript:**
1. Go to `chrome://settings/content/javascript`
2. Ensure JavaScript is allowed
3. Make sure rallies.ai isn't blocked

**Clear cache:**
1. Press `Ctrl/Cmd + Shift + Delete`
2. Select "Cached images and files"
3. Click "Clear data"

### Safari

**Enable JavaScript:**
1. Safari → Preferences → Security
2. Check "Enable JavaScript"

**Clear cache:**
1. Safari → Preferences → Privacy
2. Click "Manage Website Data"
3. Remove rallies.ai data

**Disable content blockers:**
1. Safari → Preferences → Websites
2. Click "Content Blockers"
3. Disable for rallies.ai

### Firefox

**Enable JavaScript:**
1. Type `about:config` in address bar
2. Search for `javascript.enabled`
3. Ensure it's set to `true`

**Clear cache:**
1. Press `Ctrl/Cmd + Shift + Delete`
2. Check "Cache"
3. Click "Clear Now"

### Edge

**Enable JavaScript:**
1. Go to `edge://settings/content/javascript`
2. Ensure JavaScript is allowed

**Clear cache:**
1. Press `Ctrl/Cmd + Shift + Delete`
2. Select "Cached images and files"
3. Click "Clear now"

## Extensions That May Cause Issues

These extension types can interfere with Rallies:

- **Ad blockers** - May block necessary scripts
- **Privacy extensions** - Can block data fetching
- **VPN extensions** - May cause connection issues
- **Script blockers** - Block JavaScript functionality
- **Dark mode extensions** - May conflict with our styling

### Test Without Extensions

1. Open an **incognito/private window**
2. Try Rallies there
3. If it works, an extension is likely causing issues
4. Disable extensions one by one to find the culprit

## Mobile Browser Issues

### iOS Safari
- Make sure you're running the latest iOS
- Clear Safari website data in Settings
- Disable content blockers for rallies.ai

### Chrome on Mobile
- Update Chrome via App Store/Play Store
- Clear browsing data in Chrome settings
- Consider using the Rallies app instead

## When to Use a Different Browser

If issues persist in one browser:
1. Try Chrome (our most tested browser)
2. If Chrome works, the issue is browser-specific
3. Report the issue to us with browser details
