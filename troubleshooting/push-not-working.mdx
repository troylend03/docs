# Push Notifications Not Working

Not receiving push notifications for price alerts or other updates? Here's how to fix it.

## Check Notification Settings

### In Rallies

1. Open the Rallies app or website
2. Go to **Settings → Notifications**
3. Verify push notifications are **enabled**
4. Check specific categories (alerts, earnings, etc.) are turned on

### On Your Device

Notifications need to be enabled at the device level too.

#### iPhone/iPad

1. Go to **Settings → Notifications → Rallies**
2. Make sure **Allow Notifications** is ON
3. Check that these are enabled:
   - Lock Screen
   - Notification Center
   - Banners
4. Verify **Sounds** and **Badges** if you want those

#### Android

1. Go to **Settings → Apps → Rallies → Notifications**
2. Make sure notifications are **enabled**
3. Check individual notification categories
4. Ensure **Do Not Disturb** isn't blocking them

#### Web (Desktop)

1. Click the **lock icon** in your browser's address bar
2. Find **Notifications**
3. Set to **Allow**
4. Make sure browser notifications are enabled system-wide

## Common Issues & Fixes

### "I never got asked to enable notifications"

**Mobile:**
1. Delete and reinstall the app
2. When prompted, tap **Allow** for notifications

**Web:**
1. Clear site data for rallies.ai
2. Refresh the page
3. You should see a notification permission prompt

### Notifications were working but stopped

1. **Check app updates** - Update to the latest version
2. **Check OS updates** - Some updates reset permissions
3. **Re-enable in settings** - Toggle off and on again
4. **Reinstall the app** - Fresh install can fix issues

### Getting some notifications but not others

1. Go to **Rallies Settings → Notifications**
2. Check each category individually:
   - Price alerts
   - Earnings alerts
   - AI Arena updates
   - Account notifications
3. Enable the ones you're missing

## Device-Specific Troubleshooting

### iPhone

**Check Focus modes:**
1. Go to **Settings → Focus**
2. Check if a Focus mode is active
3. Add Rallies to "Allowed Notifications" for your Focus modes

**Check notification summary:**
1. Go to **Settings → Notifications → Scheduled Summary**
2. If enabled, Rallies notifications may be bundled and delayed
3. Remove Rallies from summary for immediate delivery

### Android

**Check battery optimization:**
1. Go to **Settings → Apps → Rallies → Battery**
2. Select **Unrestricted** or **Don't optimize**
3. Background restrictions can prevent notifications

**Check Do Not Disturb:**
1. Go to **Settings → Sound → Do Not Disturb**
2. Add Rallies as an exception if needed

### Samsung Devices

Samsung has extra settings:
1. **Settings → Apps → Rallies → Mobile data**
2. Enable **Allow background data usage**
3. **Settings → Device care → Battery → App power management**
4. Add Rallies to **Apps that won't be put to sleep**

### Pixel/Stock Android

1. **Settings → Apps → Rallies → Notifications**
2. Ensure **Show notifications** is on
3. Check **Notification dot** and **Sound** preferences

## Web Push Notifications

### Chrome
1. Go to `chrome://settings/content/notifications`
2. Make sure rallies.ai is in the "Allowed" list
3. If blocked, remove and re-enable

### Safari
1. Safari → Settings → Websites → Notifications
2. Find rallies.ai and set to "Allow"

### Firefox
1. Go to `about:preferences#privacy`
2. Scroll to Permissions → Notifications
3. Ensure rallies.ai is allowed

### Edge
1. Go to `edge://settings/content/notifications`
2. Ensure rallies.ai is allowed

## Test Notifications

### Send a Test Alert

1. Go to **Settings → Notifications**
2. Look for **"Send Test Notification"** (if available)
3. Tap it and verify you receive it

### Create a Test Price Alert

1. Create a price alert for a stock that will trigger immediately
2. Set it just above or below the current price
3. Wait for it to trigger
4. If no notification, there's an issue
